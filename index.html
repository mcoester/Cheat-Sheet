CSS/HTML: Cheat Sheet

Display and Positioning

Property: Position

Value Description
Static Default Value of HTML-Elements. Sets the position of the elements from left to right and from top to bottom.
Relative Positions an element relative to its default static position.
Absolute The element will be ignored by the other elements. Also it positions the element relativ to its next parent element.
Fixed Fixes an element on a position that is dependent to the browser window.
Sticky A mixture of relative and fixed positioning. It sets the position of an element relatively to its default position but if you scroll down it sticks to a position that is dependent from the browser window.
